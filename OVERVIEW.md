# 项目概览 (Project Overview)

## 🎨 奇幻像素头像生成器

一个有趣的交互式像素艺术生成器，使用 Processing Python Mode 开发。

### ⚡ 快速开始

1. 安装 [Processing](https://processing.org/download)
2. 添加 Python Mode
3. 打开 `fantasy_avatar_generator.pyde`
4. 运行程序
5. 点击鼠标生成头像！

### 🎲 可生成组合数

**1,680,000** 种不同的头像组合！

```
2 性别 × 5 种族 × 8 肤色 × 4 表情 × 6 头饰 × 5 项链 × 5 耳环 × 5 服装 × 7 背景
= 1,680,000 种组合
```

### 🎯 核心功能

| 功能 | 描述 |
|------|------|
| 🖱️ 点击生成 | 每次点击鼠标生成新头像 |
| ⌨️ 按键生成 | 按R键生成新头像 |
| 💾 保存功能 | 按S键保存头像为PNG (Pygame版) |
| 🎲 随机组合 | 自动随机选择所有元素 |
| 🌱 种子支持 | --seed参数支持可复现生成 |
| 🎨 像素风格 | 4x4像素块构成的复古风格 |
| 📐 512x512 | 高分辨率输出 (从128x128逻辑画布) |
| 🎭 多样性 | 168万种组合可能 |
| 📦 .exe打包 | 可打包为Windows独立执行文件 |

### 🔧 技术栈

**Processing版本**:
- **语言**: Python (Processing Python Mode)
- **图形**: Processing 绘图库
- **风格**: 像素艺术 / 复古游戏风格
- **交互**: 鼠标点击、按键事件

**Pygame版本**:
- **语言**: Python 3.8+
- **图形**: Pygame 2.5+
- **命令行**: argparse支持参数解析
- **打包**: PyInstaller支持.exe打包
- **风格**: 像素艺术 / 复古游戏风格
- **交互**: 鼠标点击、按键事件、保存功能
