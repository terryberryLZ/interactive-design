## 元素生成过程 (Element Generation Process)

```
generateAvatar() 函数调用顺序:
═══════════════════════════════════════

1. 随机选择组件
   ├─ 性别 (Gender)
   ├─ 种族 (Race)
   ├─ 皮肤颜色 (Skin Color)
   ├─ 表情 (Expression)
   ├─ 头饰 (Headwear)
   ├─ 项链 (Necklace)
   ├─ 耳环 (Earrings)
   ├─ 服装 (Clothes)
   └─ 背景 (Background) ← 新增

2. 绘制背景 (多种选项: 纯色/渐变/图案)

3. 绘制顺序 (从后到前):
   ├─ drawBackground() ← 最底层 (新增)
   ├─ drawClothes()
   ├─ drawHead()
   ├─ drawExpression()
   ├─ drawEarrings()
   ├─ drawNecklace()
   └─ drawHeadwear()   ← 最顶层
```

## 使用建议 (Usage Recommendations)

1. **连续点击** - 快速点击查看不同组合
2. **截屏保存** - 遇到喜欢的头像时按截屏键
3. **观察细节** - 注意每个种族的独特特征
4. **颜色搭配** - 观察不同皮肤颜色与服装的搭配
5. **创意启发** - 将生成的头像用作游戏或设计灵感
